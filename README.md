# zstrzeszow-api
Potezne API dla strony zstrzeszow.pl oraz planu lekcji ( ͡° ͜ʖ ͡°)

## Adres API
```
https://zstapi.wiktor.tech/
```

## Endpoints

##### Lista postow z glownej strony
```
/api/posts
```

##### Lista kierunkow
```
/api/profiles
```

##### Lista nauczycieli
```
/api/teachers
```

##### Plan lekcji danej klasy
```
/api/plan/{class}

Example:
/api/plan/1ct
```
